# Project 4: Spots

Project 4 of the TripleTen Software Engineering bootcamp by Casey Driver.

## Description

This project was created by Casey Driver to practice adaptive website coding through HTML, CSS, RESt APIs,
Node.js, GitHub Deployment, using GitHub and GitBash. The website has a layout that has been optimized for desktops (at 1440px) and mobile devices. The functionality of the website is an image sharing site where you can add cards to the server, like and unlike photos, edit profile picture, name, and image, and post new photos with names and links.
